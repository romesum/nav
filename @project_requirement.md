# 导航站项目需求

## 项目概述
- 使用 Vue3 + TypeScript + Vite 开发一个导航站,前端通过supabase连接后端数据库等操作
- 首页为干净的导航页，包含背景、搜索框和用户自定义快捷入口
- 导航页作为入口

## 技术栈
- Vue3
- TypeScript
- Vite
- Ant Design Vue
- Pinia
- Vue Router
- Axios
- Formik
- Supabase

## 功能需求
- 导航页
  - 赛博风格背景
  - 头部中间包含搜索框
  - 头部
  - 分类导航
  - 网站列表
  - 谷歌登录功能
- 分类页面
- 搜索页面
- 用户中心页面

## 页面说明
- 导航页：包含头部、分类导航、网站列表和登录功能
- 分类页面：展示分类下的网站列表
- 搜索页面：展示搜索结果
- 用户中心页面：展示用户信息

## 其他需求
- 响应式设计
- 美观的 UI
- 良好的用户体验
